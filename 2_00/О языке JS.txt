Программирование - это процесс создания компьютерных программ с использованием специальных языков. 

Язык программирования - это язык, который выполняет функции посредника между человеком и компьютером. 

Особенности JavaScript:
- Был сделан, чтобы сделать веб-страницы живыми. 
- Программы - скрипты, могут встраиваться в HTML и выполняться автоматически при загрузке веб-страницы.
- Скрипты распространяются и выполняются как текст. Им не нужна специальная подготовка или компиляция для запуска. 

Любое взамодействие с пользователм добавляется с помощью JS. 

Изначально JS назывался ECMAScript, популярность языка была минимальной. На тот момент времени одним из самых популярных языков был Java, в итоге переименовали ECMAScript в JavaScript, тем самым популярность языка взлетела до небес. 

Что делает JS особенным: 
- Полная интеграция с HTML/CSS
- Простые вещи делаются просто
- Поддерживается основными браузерами и включён по умолчанию

Что можно сделать с помощью JS:
- Сайты (фронтенд-часть)
- Приложения 
- Игры 

Среды выполнения JS:
- браузер
- сервер (node.js)
- любые другие устройства

JS выполняется не только в браузере, но и на сервере и на любом другом устройстве, которое имеет специальную программу, называемую движком JavaScript. 

Что умеет JS в браузере: 
- добавляет новый HTML-код на страницу, изменять существующее содержимое, модифицировать стили
- реагировать на действия пользователя, щелчки мышью, перемещение указателя, нажатия клавищ 
- отправлять сетевые запросы на удалённые серверы, скачивать и загружать файлы
- задавать вопросы пользователю и показывать сообщения
- запоминать данные на стороне клиента

CSS развивается, некоторые возможности можно реализовать на CSS, но есть ограничения и нет такой гибкости. 

На JS делают гамбургерное меню: плавное появление, добавление анимации (частично можно сделать на CSS). 

В целях безопасности с JS не всё доступно. 

Что не умеет JS в браузере: 

- Не имеет доступа к системным функциям операционной системы. 
- Не умеет запускать программы на компьютере
- Не имеет доступа к соседним вкладкам браузера 

Спецификация JavaScript - это документация к языку программирования. 

Спецификация ECMA-262 содержит самую глубокую, детальную и формализованную информацию о JavaScript. 

Новая версия спецификации появляется каждый год, в ней добавляются новые возможности и исправляются баши, которые выявлены ранее. 

https://ecma-international.org/publications-and-standards/standards/ecma-262/
